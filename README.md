# dotpass
## Czym jest dotpass?
Dotpass pozwala łatwo kontrolować dostęp pracowników do różnych drzwi w firmie.\
Aplikacja umożliwia dołączenie pracowników do firmy i nadanie im odpowiednich uprawnień pozwalających na otwieranie wybranych drzwi.\
Wszystkie próby otworzenia drzwi są zapisywanie dla zwiększonej prywatności.

## Instrukcja obsługi
Po zalogowaniu się, poproś swoją firmę o kod zaproszenia.\
Aby otworzyć drzwi, zeskanuj tag odblokowanym telefonem. **(Wymagane NFC)**

Jeśli chcesz wykorzystać dotpass w swojej firmie, skontaktuj się z nami!
[Dane kontaktowe]
## User stories
### Użytkownik może:
- Zarejestrować konto
- Zalogować się
- Dołączyć do firmy za pomocą kodu zaproszenia
- Zobaczyć listę dostępnych dla niego drzwi
- Otworzyć drzwi skanując tag
### Administrator dodatkowo może:
- Tworzyć/edytować/usuwać grupy w firmie
- Generować kod zaproszenia do firmy
- Akceptować prośby pracowników o dołączenie
- Zobaczyć listę pracowników
- Dodawać/usuwać pracowników z grup
- Usuwać pracowników z firmy
- Edytować uprawnienia grupom/pracownikom
- Zobaczyć listę połączonych płytek
- Otworzyć profil płytki
- Zmienić nazwę płytki
- Sparować płytkę z tagiem
- Zobaczyć historię dostępu do płytek
